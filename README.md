# Power-BI-Dashboard-And-Report
# 📊 Sales Dataset Analysis Dashboard

## 📌 Overview
This project presents an interactive **Sales Dashboard** designed to analyze business performance and generate actionable insights. The dashboard helps in understanding sales trends, customer behavior, and profit distribution.



## 🎯 Key KPIs
- Total Orders: 400  
- Total Profit: 400  
- Average Rating: 3.03  
- Average Review Count: 262.88  
- Maximum Value: 2.50K  
- Minimum Value: 9.75  



## 📈 Features & Insights

### 📅 Monthly Revenue Analysis
- Visualized revenue trends across months  
- Identified peak sales periods and fluctuations  

### 💳 Payment Method Analysis
- Compared profit contribution across payment methods  
- Wallet shows highest profit contribution  

### 🌍 Regional Analysis
- Sales filtered by regions: Asia, Europe, Middle East, North America  

### 🛍️ Product Category Insights
- Analyzed revenue contribution by categories (Beauty, Books, Electronics, Fashion, Home & Kitchen, Sports)  

### 📆 Year-wise Comparison
- Compared revenue performance between **2022 and 2023**  

### 🔍 Quantity vs Profit
- Scatter plot used to analyze relationship between quantity sold and total profit  



## Interactive Elements
- Product Category slicer  
- Customer Region slicer  
- Payment Method slicer  
- Year filter (2022–2023)  



## Tools Used
- Power BI  
- Excel (for dataset)
